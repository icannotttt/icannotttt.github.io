退出操作

:wq：保存并退出。
:w：保存但不退出。
:q!：强制退出而不保存。
:qa：退出所有打开的文件。
:qa!：强制退出所有打开的文件，不保存任何更改。

模式切换

进入插入模式：按 i 键。
返回普通模式：按 Esc 键。
进入可视模式：按 v 键。
进入可视行模式：按 V 键。
进入可视块模式：按 Ctrl+v 键。
进入命令行模式：按 : 键。

移动

gg：跳到文件的第一行。
G：跳到文件的最后一行。
nG：跳到文件的第 n 行。
0：跳到当前行的行首。
：跳到当前行的行尾。gg0：跳到文件的第一行的行首。G：跳到当前行的行尾。 gg0：跳到文件的第一行的行首。 G：跳到当前行的行尾。gg0：跳到文件的第一行的行首。G：跳到文件的最后一行的行尾。
Ctrl+f：向下翻一页。
Ctrl+b：向上翻一页。
Ctrl+d：向下翻半页。
Ctrl+u：向上翻半页。
Ctrl+e：向下滚动一行。
Ctrl+y：向上滚动一行。

插入文本

i：在当前光标位置插入文本。
I：在当前行的行首插入文本。
a：在当前光标位置的后面插入文本。
A：在当前行的行尾插入文本。
o：在当前行的下面插入一行。
O：在当前行的上面插入一行。

删除行

dd：删除当前行。
ndd：删除从当前行开始的 n 行。
d0：删除从当前光标位置到行首的内容。
d$：删除从当前光标位置到行尾的内容。

复制 粘贴

复制：
yy：复制当前行。
nyy：复制从当前行开始的 n 行。
y0：复制从当前光标位置到行首的内容。
y$：复制从当前光标位置到行尾的内容。
粘贴：
p：在当前光标位置的后面粘贴。
P：在当前光标位置的前面粘贴。

查找和替换

查找：
/pattern：向下查找匹配 pattern 的内容。
?pattern：向上查找匹配 pattern 的内容。
n：查找下一个匹配项。
N：查找上一个匹配项。
替换：
:s/old/new/：在当前行替换第一个匹配的 old 为 new。
:s/old/new/g：在当前行替换所有匹配的 old 为 new。
:%s/old/new/g：在文件中替换所有匹配的 old 为 new。

撤销和重做

撤销：
u：撤销上一次操作。
重做：
Ctrl+r：重做上一次撤销的操作。

窗口操作

分割窗口：
:split 或 :sp：水平分割窗口。
:vsplit 或 :vsp：垂直分割窗口。
切换窗口：
Ctrl+w 然后按方向键（h、j、k、l）：在窗口之间切换。
关闭窗口：
:close 或 :clo：关闭当前窗口。
:only 或 :on：关闭所有其他窗口，只保留当前窗口。
